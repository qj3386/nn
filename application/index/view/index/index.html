“云养殖”的发展

由虚拟转为现实

随着阿里巴巴在支付宝客户端的公益项目蚂蚁森林兴起，内蒙古阿拉善盟地区的荒漠里种下了76万棵梭梭树，从太空往下看，一片赤黄的图片，带上了点点的绿！

这在以前看来是多么不可思议的事啊！每个人捐献的树种甚至还有编号，对应着荒漠里的一颗树苗！


这就是云种植。
